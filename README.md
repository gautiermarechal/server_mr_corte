# Mr. Corte Barbershop - Booking System
Here is the back-end of the booking appointment system for the Montreal barbershop Mr. Corte. This communicates with a MongoDB database that you run on your local machine.
You can find the front-end of the project here: https://github.com/gautiermarechal/booking-appointment-system-m-corte

# How to run it?

### Download the zip file

### `cd server_mr_corte`

### `npx nodemon index.js`
The server will run on your localhost:3000
